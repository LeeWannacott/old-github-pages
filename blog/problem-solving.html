Solved a problem in an elegant way: 


        // no split() ,or filtering() required by using replace() and a regular expression. 
        // (g) means look globally for any bees in the string
        // the (i) means don't worry about case sensitivity. 
        function deBee (str){ 
          beeLessString = str.replace(/bees/gi,"") 
          return beeLessString 
        }
