What does it mean to display inline vs inline blocks?

Example: display:inline

Element one. Element two.

Example: display:inline-block

Block one Block two.

The main difference is display: inline-block allows width and height to be set on an element.

inline-block respects top and bottom margins/padding. display: inline does not.

Example: display:block

Block one Block two.

display: block the difference is that display: inline-block does not add a line-break after the element, so the element can sit next to other elements.
