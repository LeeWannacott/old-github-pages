What does it mean to display inline vs inline blocks?

Example: display: inline;

Element one. Element two.

Show elements next to each other horizontally.

Example: display: inline-block;

Block one Block two.

The main difference is display: inline-block allows width and height to be set on an element.

display: inline-block respects top and bottom margins/padding.

display: inline-block does not add a line-break after the element; so the elements can sit next to other elements hence the term: inline.

Example: display: block;

Block one Block two.

display:block can be used to have images stacked on top of each other.

display:block as demonstrated by being on the left hand side of the page doesn't text align: center.
